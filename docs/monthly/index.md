# 月別アーカイブ

このページでは、各月にマージされたRuby on RailsのPull Requestを月別に閲覧できます。

## アーカイブ一覧

サイドバーから各月のダイジェストにアクセスできます。最新の月が上部に表示されます。

---

各月のページでは、以下の情報を確認できます：

- **PR番号とタイトル**: マージされたPRへのリンク
- **マージ日**: PRがマージされた日付
- **作成者**: PRの作成者
- **AI要約**: OpenAI GPT-4oによる日本語での要約と解説
  - 概要
  - 変更内容の詳細
  - 影響範囲・注意点
  - 参考情報

## 検索機能

ページ上部の検索バー（またはキーボードショートカット `/`）を使用して、過去のPRを検索できます。
